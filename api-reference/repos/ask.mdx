title: 'Ask'
api: 'POST https://api.relace.ai/v1/repo/{repo_id}/ask'
description: 'Ask natural language questions about repo content'
---

## Path Parameters

<ParamField path="repo_id" type="string" required>
  Repo ID
</ParamField>

## Request Body

<ParamField body="query" type="string" required>
  Natural language question about the repo
</ParamField>

## Response

<ResponseField name="answer" type="string">
  Natural language answer generated by the AI based on repo content
</ResponseField>


<RequestExample>

```bash cURL
curl -X POST https://api.relace.ai/v1/repo/123e4567-e89b-12d3-a456-426614174000/ask \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "How does the user authentication system work?"
  }'
```

```python Python
import requests

repo_id = "123e4567-e89b-12d3-a456-426614174000"
url = f"https://api.relace.ai/v1/repo/{repo_id}/ask"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "query": "How does the user authentication system work?"
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
```

</RequestExample>
